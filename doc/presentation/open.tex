\section*{
\vskip 1em
ABINIT : 2. OPEN SOURCE }

\vskip 1em
\parbox{0.92\linewidth}{
\Large{
\begin{itemize}
 \item
 Is the use of ABINIT subject to an agreement ? 
 \item
 Are the sources available ? 
 \item
 Could I modify them, and redistribute them ?
\end{itemize}

\vskip 1em
 Starting with ABINITv3.0 (released in December 2000),
 the distribution of ABINIT is under GNU 
 {\red General Public Licence} (GPL), 
 see http://www.gnu.org/copyleft/gpl.html .
\vskip 1em
 {\it{``The licences for most softwares are designed 
     to take away your freedom to share and change it. 
     By contrast, the GNU GPL is intended to guarantee 
     your freedom to share and change free software -- 
     to make sure the software is free for all its users.''}}
\vskip 2em
You have the freedom to :
\begin{itemize}
 \item[\red \ding{229}]
  run the program, for any purpose
 \item[\red \ding{229}]
  modify the program to suit your needs
 \item[\red \ding{229}]
  redistribute copies
 \item[\red \ding{229}]
  distribute modified versions (if they are advertised to be so)
\end{itemize}
\vskip 1em
 ... Others have the same rights as you ! \\
 ... If one delivers binaries one must also deliver the sources !\\
 ... Can be used in another package only if that package is GNU GPL !\\
\vskip 1em
{\blue GNU GPL acts like a computer virus. \\
       Sharing of code is guaranteed.}
}}
